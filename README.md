> A CLI tool for getting stats of NPM packages

### 🏠 [Homepage](https://www.npmjs.com/package/getstat)

## Install

```sh
npm install getstat -g
```

## Usage

```sh
getstat<package name>
```

## Example

```sh
getstat react -d
```


## Run tests

```sh
npm run test
```
